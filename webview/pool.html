API Worker chuyển tiếp cục bộ

Worker cục bộ luôn cố định tại 127.0.0.1:9182. Bạn chỉ cần nhập API URL, Token và Model thật ngay tại đây, extension sẽ lưu vào file local để IDE tự động chuyển tiếp request đúng đích, không cần mở trang xác thực cũ nữa.

Cấu Hình API Đích

Nhập API URL, Token và Model của dịch vụ AI hoặc gateway bạn muốn sử dụng. Sau khi bấm lưu, extension sẽ ghi cấu hình vào file của local worker để mọi request từ IDE được chuyển tiếp đúng endpoint. Nếu bạn đã đổi endpoint mà request vẫn chạy sang cấu hình cũ, hãy khởi động lại api-worker một lần.

Quy Trình Kích Hoạt

B1: chạy api-worker. B2: lưu cấu hình API ở trên. B3: kích hoạt Cursor để sử dụng cơ chế chuyển tiếp cục bộ. Nếu sau khi lưu mà request vẫn đi tới upstream cũ, hãy tắt và chạy lại local worker một lần rồi làm mới trạng thái.
